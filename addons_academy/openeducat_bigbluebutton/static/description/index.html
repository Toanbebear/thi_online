System Parameter of Bigbluebutton url key

Go to menu Settings -> Technical -> Parameters -> System Parameters. Click on Create button to create new url key

System parameter of Bigbluebutton Secret key

Go to menu Settings -> Technical -> Parameters -> System Parameters. Click on Create button to create new Secret key.

Now, Go to menu Timetables -> Sessions. Here Select Particular Session on which you want to create meeting.

Click on button Create online meeting, wizard will be opened.

Click on button Create meeting. Online meeting is started as shown in image.
